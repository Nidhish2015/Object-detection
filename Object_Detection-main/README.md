# YOLOv8 IP Webcam Object Detection

This project performs real-time object detection using YOLOv8 and a mobile IP Webcam.

## Features
- Real-time detection
- Uses phone camera
- Lightweight YOLOv8 model

## Requirements
- Python 3.x
- OpenCV
- Ultralytics

## Run
```bash
python object_detection.py
